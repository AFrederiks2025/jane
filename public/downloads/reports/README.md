# Voorbeeldrapportages

Plaats hier de drie PDF-bestanden van de Jane® voorbeeldrapportages. De
methodiek-pagina (`/nl/talenten-methodiek`) verwacht ze op deze exacte
namen:

- `jane-talenten-rapportage-nl.pdf` — de standaard Jane® Talenten Rapportage
- `jane-functie-rapportage-nl.pdf` — de Jane® Functie Rapportage
- `jane-jong-talenten-rapportage-nl.pdf` — de Jane® Jong Talenten Rapportage

Bron-bestanden zoals aangeleverd:

- `Jane-Talenten-Rapportage-voorbeeldkandidaat.pdf` → hernoem naar
  `jane-talenten-rapportage-nl.pdf`
- `Jane-Functie-Rapportage-voorbeeldkandidaat.pdf` → hernoem naar
  `jane-functie-rapportage-nl.pdf`
- `Jane-Jong-Talenten-Rapportage-voorbeeldkandidaat.pdf` → hernoem naar
  `jane-jong-talenten-rapportage-nl.pdf`

Zodra de bestanden hier staan, opent de "Download"-knop op de
methodiek-pagina ze in een nieuw tabblad. De thumbnail boven elke kaart
wordt in code gerenderd (zie [`src/components/ReportCover.tsx`](../../../src/components/ReportCover.tsx))
en is dus al zichtbaar voordat de PDF's er staan.
