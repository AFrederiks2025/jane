# docs/ — bronmateriaal (niet publiek)

Zet hier bronbestanden die ik (Claude) moet kunnen lezen, maar die **niet**
op de website getoond worden. Alles in deze map staat wél in de Git-repo,
maar wordt niet door de website geserveerd (anders dan `public/`).

## Waar zet ik wat?

| Soort bestand | Map |
|---|---|
| Drafts per doelgroep (de 13 Word-bestanden) | `docs/doelgroepen/` |
| Overig bronmateriaal (oude presentaties, boeken, cases, PDF's ter referentie) | `docs/bronmateriaal/` |
| PDF's die bezoekers moeten kunnen **downloaden** (rapportages, brochures) | `public/downloads/` — NIET hier |

## Tip

- Word-bestanden (`.docx`), PDF's en tekst kan ik allemaal lezen.
- Geef bestanden een duidelijke naam, bv. `Landingspagina — Voorgangers.docx`.
- Na het toevoegen kun je mij gewoon vragen "verwerk de drafts in docs/doelgroepen"
  en dan zet ik ze om naar de pagina's.
